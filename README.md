## Descripción

Datos económicos completos sobre la **carga económica global de los trastornos mentales**, con énfasis en América Latina y el Caribe. Datos de OMS, Banco Mundial, OPS, PNUD y Deloitte (2020-2024).

## Cifras clave

| Métrica | Valor | Fuente |
|---|---|---|
| Pérdida anual global de productividad | **1 billón USD/año** | OMS |
| Pérdidas globales PIB 2019 (AVAD) | **4,7 billones USD** | OMS/PMC |
| Pérdidas acumuladas 2011-2030 | **16,3 billones USD** | Comisión Lancet |
| Costo global acumulado 2040 | **14 billones USD** | Deloitte |
| AVAD atribuibles (2019) | **418 millones** | OMS |
| Sudamérica 2020-2050 | **7,3 billones USD** | OPS |
| Brasil 2020-2050 | **3,7 billones USD** | OPS |
| Colombia depresión (2015) | **840 millones USD** | Banco Mundial |
| 1 de cada 4 personas en ALC | con salud mental deficiente | OMS |
